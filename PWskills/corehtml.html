List of some Html tags

  1. < Html > : Represents the root of an HTML document.
  2. < Title > : Specifies the title of the HTML document (displayed in the browser's title bar or tab).
  3. < Head > : Contains metadata about the HTML document.
  4. < Body > : Represents the content of an HTML document
  5. < H1>-< H6> : Heading Tags in different sizes.
  6. < p> - Represents a paragraph of text.
  7. < a> - Creates a hyperlink to another webpage or resource.
  8. < img> - Embeds an image into the HTML document.
  9. < ul> - Defines an unordered (bulleted) list.
  10. < ol> - Defines an ordered (numbered) list.
  11. < li> - Represents a single item in a list.
  12. < div> - Defines a division or a section in an HTML document.
  13. < span> - Inline element used to group elements and apply styles.
  14. < table> - Represents tabular data.
  15. < tr> - Defines a row in a table.

Fullstack Web Devlopment

  1. Front-end
    • HTML/CSS: HTML (HyperText Markup Language) is the standard markup language for creating web pages. CSS (Cascading Style Sheets) is a style sheet language used for describing the look and formatting of a document written in HTML.
    • JavaScript (JS): A high-level, interpreted scripting language for creating and controlling dynamic website content.
    • Front-end Libraries / Frameworks: Examples include React.js, Vue.js, and Angular.js. These help organize code, create user interfaces, and manage website interactivity.
  2. Back-end
    • Node.js: A JavaScript runtime environment that executes JavaScript code outside of a browser.
    • Express.js: A web application framework for Node.js, designed for building web applications and APIs.
    • Java: A high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible.
  3. Databases
    • SQL Databases (Relational): Examples include MySQL, PostgreSQL, and Oracle. These databases store data in tables and rows, making it easier to find and organize information.
    • NoSQL Databases (Non-Relational): Examples include MongoDB, CouchDB, and Cassandra. These databases can handle unstructured data and can scale horizontally.
  4. Version Control Systems:
    • Git: A free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.
  5. Deployment:
    • Docker: An open-source platform used for automating the deployment, scaling, and management of applications.
    • AWS (Amazon Web Services): Offers reliable, scalable, and inexpensive cloud computing services.
